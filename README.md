## WebVR Movies Info

Web VR Trending/Popular Movies Information using React, A-frame
[A-Frame](https://aframe.io) and React with
[aframe-react](https://github.com/ngokevin/aframe-react).



### Installation

To get started:

```bash
npm install
npm start
```

To publish to GitHub Pages:

```bash
npm run publish
```
